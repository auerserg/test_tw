# Вопросы

## Як працюють індекси у БД?

Иденксы работаю путем создания структуры данных, которая упорядочивает информация в определенном порядке для ускорения операция сортировки, поиска и фильтрации. С помощью процесса индексирования не требуется проверять всю таблицу, а можно использовать индецсы для быстрого поиска нужных данных.

## Які є JOIN в MySQL?
 * **INNER JOIN** - возвращает строки с обеих таблиц соответствующие условию
 * **LEFT JOIN** - возвращает строки с левой таблице и строки с правой соответствующие условию
 * **RIGHT JOIN** - возвращает строки с правой таблице и строки с левой соответствующие условию
 * **FULL JOIN** - возвращает строки с обеих таблиц в том числе строки которые не имеют соответсвия в другой таблице
 * **SELF JOIN** - обьединенние таблицы самой с собою.


## Що таке dependency inversion?

Это принцип проектирования ПО, направленный на уменьшение связаности между компонентами системы и улучшения их гибкости и переиспользуемости.

## У чому різниця між GET та POST запитом?

Они отличаются:
 * Целью - `GET` используется для получения данных с сервера, средствами передачи некоторых параметров. `POST` для отправки данных на сервер, с целью их измененния или обработки.
 * Параметрами - `GET` запрос передает параметры в url, как часть строки запроса. `POST` передает параметры в теле запроса (в url параметры не видны)
 * Безопасность - `GET` параметры так как передаются в url их видно и легко манипулировать ими. `POST` передает параметры в теле запроса что можно считать их скрытыми.
 * Кэширование - `GET` параметры могут быть легко закешированы. `POST` параметры невозможно заккешировать
 * Закладки - `GET` запрос можно легко сохранить в закладка и повторно использовать для получения тех же данных или более новых.
 * Длинной данных - `GET` параметры ограничиваются макимальной длинной строки запроса. `POST` параметры практически не ограничены по обьему данных (может быть ограничен настройками сервера)

## У чому різниця дзвінка self і static?

`self` привязан к класу где он и вызывается, когда `static` вызывает свойства к текущему классу с привязкой к наследованию в котором происходит вызов.

## Яка різниця між MVP та MVC?

В **MVP** презентер отдельно управляет взаимодействием с представлением, решая, какие данные предоставлять и как обрабатывать действия пользователя.
В **MVC** контроллер имеет больше ответственности за управление представлением, решая, какие данные отображать и как реагировать на события.

## Яка різниця між == та ===?

Оператор сравненния `==` используется для сравненния значения двух переменных без учета типа данных. К примеру: 1 == '1' - будет правда
Оператор идентичности `===` сравнивает значенния с учетом типа данных. К примеру: 1 === '1' - будет ложь

## Що таке final class та final method?

`final class` - обьявляется как класс, который невозможно расширить другим классом. То б то невозможно создать класс которые использовать extends для финально класса
`final method` - определяет что метод невозможно переопределить в подклассах.
